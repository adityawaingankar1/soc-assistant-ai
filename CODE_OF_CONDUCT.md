# Code of Conduct

## Our Standards

We are committed to providing a welcoming and respectful environment.

Examples of positive behavior:

- Be respectful.
- Be constructive.
- Help others learn.
- Accept feedback professionally.

Unacceptable behavior includes:

- Harassment
- Personal attacks
- Hate speech
- Sharing confidential information

## Enforcement

Project maintainers may remove contributions that violate this Code of Conduct.
