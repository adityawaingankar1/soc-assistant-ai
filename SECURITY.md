# Security Policy

## Supported Versions

This project is actively maintained for the latest release.

## Reporting Security Issues

Please do not open public GitHub issues for security vulnerabilities.

Instead, contact:

adityawaingankar1@gmail.com

Include:

- Description
- Steps to reproduce
- Potential impact

Sensitive reports will be handled confidentially.
